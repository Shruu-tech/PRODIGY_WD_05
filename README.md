# Weather App ⛅

# Description  
This project is a **Weather App** that fetches real-time weather data using an API based on the user's location or a manually inputted location.

# Features  
✅ Fetch weather data using an API  
✅ Display temperature, conditions, and more  
✅ User-friendly UI  
✅ Supports manual location input
